><summary>约定</summary>
>
>1. `+`、`-`含义
>
>    兼容ie8+：包括ie8以及高于ie8的ie浏览器；兼容ie8-：包括ie8以及低于ie8的ie浏览器。
>2. 变量命名含义
>
>    1. `dom`为JS对象，`$dom`为jQuery（或Zepto）对象。
>    2. `obj`为对象，`arr`为数组。
>3. 默认仅针对浏览器环境的内容
